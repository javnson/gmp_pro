为了让用户清晰明了，工业界通用的规范是“逆变器视角 (Inverter Convention)”。我们应当在模块的 Doxygen 注释、用户手册甚至变量命名中，严格贯彻以下标定法则：

#### 1. 物理传感器的极性标定（必须在底层硬件抽象层对齐）

- **电压 ($V_{grid}$)**：电网火线 (L) 相对零线 (N) 为正时，ADC 读取的标幺值为正。
- **电流 ($I_{grid}$)**：电流**从逆变器流向电网**（流出）时，ADC 读取的标幺值为正。

#### 2. 功率与能量的极性标定

- **有功功率 ($P > 0$)**：系统处于**逆变模式 (Inverting)**。直流母线正在失去能量（放电），电网正在获得能量。
- **有功功率 ($P < 0$)**：系统处于**整流模式 (Rectifying/AFE)**。直流母线正在获取能量（充电），电网正在失去能量。
- **无功功率 ($Q > 0$)**：系统发出滞后无功（**呈感性**）。
- **无功功率 ($Q < 0$)**：系统发出超前无功（**呈容性**）。

#### 3. 直流母线电压外环的“反逻辑” (极易出错的细节)

如果用户使用电压 PI 控制器来维持直流母线电压（比如目标是 $400\text{V}$）：

- 当母线电压跌落（例如只有 $380\text{V}$），误差 $e = 400 - 380 = +20\text{V}$。
- 此时系统需要从电网“吸取”能量来补充电压。
- 根据我们的极性标定，吸取能量意味着 $P_{ref}$ 必须是**负数**。
- **结论：** 维持直流母线电压的 PI 控制器，其输出结果必须取反（或者 PI 的输入误差定义为 $V_{fbk} - V_{ref}$）才能作为 $P_{ref}$ 喂给指令发生器！
