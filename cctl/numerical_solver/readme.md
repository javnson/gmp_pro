# CCTL numerical solvers

**English** | [简体中文](readme_cn.md)

This directory provides C++ numerical solvers for simulation or host environments with comparatively generous memory. A solver may use lower-level mathematical operations implemented in C.

- `solver_base.hpp` defines the common solver interface.
- `runge_kutta_4.hpp` implements a fourth-order Runge–Kutta solver.
